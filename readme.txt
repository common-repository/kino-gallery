The Kino Gallery adds a clickable thumbnail gallery to your sidebar.



== Description ==

For full instructions on how to use the Kino Gallery plugin please refer the [Installation](http://wordpress.org/extend/plugins/kino-gallery/installation/ "Installation") section.

The Kino Gallery enables you to easily setup a thumbnail gallery that sits in the sidebar. When a thumbnail is clicked the main image will be loaded up into the main column of the website. Titles and fully featured descriptions can be added to these images.

The Kino Gallery is a free to use plugin donated to the community by Kino Creative. 

If you use the plugin please rate it.


== Installation ==

= Install Kino Gallery = 
1. Upload the plugin folder, kino-gallery, to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Alternatively, simply run a search for Kino Gallery in the Plugins section and follow onscreen installation instructions.

= Setup plugin on site = 

1. Drag the Kino Gallery widget into the sidebar and give it a name.
2. In Settings>Kino Gallery you can change; Max width of full size image, Thumbnail size, Thumbnail CSS (where you can apply custom styling), Opacity of thumbnails, Fade speed of thumbnails
3. Add new images to the sidebar by going to Kino Gallery>Add New. Note that you just need the URL of the large image here as the plugin will resize the thumbnail according to the settings.
4. You can edit the gallery through Kino Gallery>Kino Gallery
